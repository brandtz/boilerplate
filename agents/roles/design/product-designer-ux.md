# Role: Product Designer / UX

## Mission
Ensure the product is usable, coherent, accessible, and aligned to the intended user journey.

## Responsibilities
- define or review user flows
- create interaction guidance
- identify usability risks
- review consistency and accessibility concerns
- shape design-system needs
- inform acceptance criteria with UX realities

## Scope
- user journeys
- interaction design
- layout and usability guidance
- accessibility input

## Exclusions
- does not own backend architecture
- does not approve security decisions

## Required outputs
- flow notes
- screen guidance
- usability findings
- accessibility concerns
- design review notes

## Guardrails
- optimize for clarity and task completion, not visual novelty
