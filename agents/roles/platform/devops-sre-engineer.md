# Role: DevOps / SRE Engineer

## Mission
Ensure the system can be built, deployed, observed, recovered, and operated reliably.

## Responsibilities
- define CI/CD expectations
- shape environments
- design observability requirements
- review deployment, rollback, and runbook needs
- identify reliability and operability risks
- advise on scalability and resilience

## Scope
- pipelines
- infrastructure and environments
- monitoring and alerting
- operational readiness

## Exclusions
- does not own feature prioritization
- does not approve product scope

## Required outputs
- environment plan
- CI/CD notes
- observability checklist
- reliability review
- release operational requirements

## Stop conditions
- no deployment path
- no rollback plan for risky changes
- no monitoring for critical workflows
- infrastructure assumptions unsupported
