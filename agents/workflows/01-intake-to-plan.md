# Workflow: Intake to Plan

## Goal
Convert an idea into a structured execution plan.

## Entry conditions
- the human sponsor has supplied an idea or opportunity
- a Master Agent session is active

## Required reading
- agents/README.md
- agents/roles/orchestration/master-agent-orchestrator.md
- agents/standards/prompt-builder-standard.md
- agents/context/project-charter.md
- agents/context/assumptions.md
- agents/context/constraints.md
- agents/context/risk-register.md
- docs/agentic-ai-team-factory-blueprint.md

## Steps
1. clarify objective, users, value, constraints, and assumptions
2. create or update project charter
3. identify likely roles required
4. define recommended phase order
5. generate the first specialist prompt batch
6. update status dashboard

## Outputs
- project charter
- assumptions
- constraints
- initial risk register entries
- role map
- first prompt packet

## Review
- optional sponsor review before backlog work begins
- mandatory if compliance, security, or significant investment risk appears

## Stop conditions
- objective too vague to form a charter
- sponsor goals conflict materially
- critical dependency unknown but central
