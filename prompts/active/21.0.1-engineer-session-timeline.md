## Prerequisites

## Context for the Agent

You are the Senior Software Engineer. Build the session timeline/history view showing all session handoffs chronologically, and implement prompt-to-handoff linking with changed file display.

## Required Reading

```text
docs/technical-tasks-e2-e5-ui.md
docs/dashboard-ux-wireframes.md
```

## Scope

- Build session timeline UI component (chronological list of sessions)
- Each entry: session ID, prompt number, role, date, summary, changed file count, status outcome
- Implement expand/collapse for full session detail
- Implement prompt-to-handoff linking in UI (from prompt detail → matching handoff)
- Display changed files list in session detail
- Flag prompts with missing handoffs as warnings
- Write component tests for timeline rendering and expand/collapse

## Constraints

- Timeline must handle 100+ session entries performantly
- Missing handoffs should show a warning, not an error

## Validation Commands

```bash
cd apps/dashboard && npm test -- --testPathPattern="session|timeline|handoff"
```

## Handoff Instructions

Create handoff notes. Include: components built, linking approach, test results.
