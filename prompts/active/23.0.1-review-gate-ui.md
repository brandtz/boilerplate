## Context for the Agent

You are the Master Agent Orchestrator running a code review gate for the UI implementation (prompts 16.0.1–22.0.1). Route through Architect, QA, Security, and UX review. Collect findings and approve or request revision.

## Required Reading

```

## Scope

- Architect review: Does UI match ADR-003? Is component structure clean?
- QA review: Is test coverage adequate? Are interaction tests present?
- Security review: Is markdown rendering XSS-safe? Are paths sanitized?
- UX review: Does implementation match wireframes? Are accessibility requirements met?
- Collect all findings into a review report
- Approve or request rework (use inserted prompts 23.0.2+ for rework)

## Constraints

- Do not modify implementation code directly
- Use inserted prompts for any needed rework

cd apps/dashboard && npm run build && npm test -- --coverage
```
