- Create test fixtures: missing fields, invalid types, empty frontmatter, no frontmatter, partial data
- Write tests verifying parser resilience (no crashes)
- Write tests verifying warning output for each invalid case
- Write tests verifying partial data extraction where possible
- Verify dashboard UI handles warnings display correctly
