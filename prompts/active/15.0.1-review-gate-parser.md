## Context for the Agent

You are the Master Agent Orchestrator running a code review gate. Route the parser implementation (prompts 11.0.1–14.0.1) through Architect, QA, and Security review. Collect findings, determine if rework is needed, and approve or request revision.

## Scope

- Architect review: Does implementation match ADR-002? Are interfaces clean?
- QA review: Is test coverage adequate? Are edge cases covered?
- Security review: Are file paths sandboxed? Any injection vectors?
- Collect all review findings into a review report
- Approve or request rework (if rework needed, create inserted prompts 15.0.2+)

## Constraints

- Do not modify implementation code directly
