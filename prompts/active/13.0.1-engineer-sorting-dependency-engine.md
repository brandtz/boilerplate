## Context for the Agent

You are the Senior Software Engineer. Implement the natural prompt sequence sorting (numeric tuple sort for `major.branch.revision` IDs) and the dependency graph / eligibility engine that determines next-runnable prompts and downstream impact of insertions.

## Scope

- Implement numeric tuple comparison for prompt IDs (major.branch.revision)
- Implement prerequisite resolution against prompt status map
- Implement next-prompt selection algorithm (ready + prerequisites met + lowest sort)
- Implement downstream impact detection for inserted prompts
- Write sort tests with edge cases (insertions, branches, gaps, non-sequential IDs)
- Write integration tests with a multi-prompt dependency graph

## Constraints

- Sorting must use numeric comparison, never string comparison
- Eligibility engine must handle circular dependencies gracefully (warn, don't infinite-loop)
