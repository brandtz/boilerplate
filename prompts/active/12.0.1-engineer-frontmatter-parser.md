## Context for the Agent

You are the Senior Software Engineer. Implement the YAML frontmatter parser for prompt and handoff files. Build the directory scanner, frontmatter extractor, validators, and the normalized graph model builder. Include comprehensive unit tests.

## Scope

- Implement YAML frontmatter extraction using gray-matter
- Implement directory scanner for prompts/, agents/epics/, agents/handoffs/
- Implement frontmatter validators for prompt and handoff contracts
- Build the normalized graph model (epic → story → task → prompt → handoff)
- Handle malformed files gracefully (emit warnings, parse what is possible)
- Create test fixtures: valid prompts, valid handoffs, malformed files
- Write unit tests for each parser component

## Constraints

- Parser must handle missing frontmatter (file with no YAML block)
- Parser must handle partial frontmatter (missing optional fields)
- Parser must not throw on any valid markdown file
