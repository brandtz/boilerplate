## Context for the Agent

You are the Master Agent Orchestrator running the final release readiness gate. Verify all epics are complete, all review findings are resolved, all tests pass, and the project meets acceptance criteria from the PRD. Present go/no-go to the human sponsor.

## Scope

- Verify all 6 epics are complete or have documented deferrals
- Verify all acceptance criteria from PRD Section 18 are met
- Verify all tests pass with adequate coverage
- Verify all security findings are resolved or mitigated
- Verify accessibility audit findings are resolved
- Complete release readiness checklist
- Update status-dashboard.md
- Present final go/no-go recommendation
- **Propagate findings:** If any non-blocking findings or recommendations are identified, update the next downstream prompt (29.0.1) to include them as pre-work, or create an inserted prompt (28.0.2) if the scope warrants separation. All review findings are pre-approved for implementation — the executing agent of the downstream prompt should work them without waiting for additional approval.

## Constraints

- Human sponsor must approve before release
- Any unresolved critical or high-severity findings block release
- All low/medium findings must be propagated to the immediate downstream prompt as actionable tasks (never left as untracked recommendations)

```bash
cd apps/dashboard && npm run build && npm test -- --coverage
```

2. **Update this prompt's frontmatter:**
   - Set `status: "done"`
   - Set `session_handoff` to the handoff file path created in step 1
   - Set `completed_at` to the current ISO-8601 timestamp
   - Set `updated_at` to the current ISO-8601 timestamp

3. **Update `prompts/index.md`:**
   - Change this prompt's row: status to `done`, add handoff path, set completed date, add summary note
   - Update the summary counts (decrement ready, increment done)

4. **Update `agents/context/status-dashboard.md`:**
   - Add this prompt to Active Work Packets as **DONE** with a brief note
   - Update Metrics counts (Prompts Completed, ready/done tallies)
   - Update Next Suggested Action to the next prompt in sequence
   - Update Awaiting Review if any items were resolved or added

   - Add a row for each decision made during this session
