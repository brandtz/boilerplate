## Context for the Agent

You are the Senior Software Engineer. Implement the JSON state emitter that serializes the normalized graph model into the `dashboard-state.example.json` schema. Create a CLI entry point that accepts a repo path and outputs the JSON file.

## Scope

- Implement JSON serialization of DashboardState
- Conform output to the schema defined in dashboard-state.example.json
- Create CLI entry point: `npx dashboard-parse --repo <path> --output <json-path>`
- Write snapshot tests comparing emitter output to expected JSON
- Test with the actual repo as input (self-referential test)

## Constraints

- Output must be deterministic (same repo = same JSON)
- CLI must validate repo path exists and is dashboard-capable before parsing

## Validation Commands

