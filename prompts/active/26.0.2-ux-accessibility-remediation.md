## Scope

### HIGH priority fixes (WCAG Level A):

1. **F4 — Global focus-visible indicators**
   - Add `*:focus-visible { outline: 2px solid #3B82F6; outline-offset: 2px; }` to `globals.css`
   - File: `src/app/globals.css`

2. **F2 — Skip-to-content link**
   - Add visually hidden skip link as first child of `<body>` in `layout.tsx`
   - Add `id="main-content"` to `<main>` in `AppShell.tsx`
   - Files: `src/app/layout.tsx`, `src/components/shell/AppShell.tsx`

3. **F3 — Keyboard-accessible sort headers**
   - Add `tabIndex={0}`, `role="columnheader"`, `onKeyDown` (Enter/Space) to `<th>` elements
   - Add `aria-label` describing sort action
   - File: `src/components/prompts/PromptTable.tsx`

4. **F1 — Chart accessible alternatives**
   - Wrap each chart in `<figure>` with `role="img"` and `aria-label`
   - Add visually hidden `<table>` with raw data below each chart
   - Files: `src/components/overview/EpicCompletionChart.tsx`, `PromptStatusChart.tsx`, `SessionThroughputChart.tsx`, `RemainingPromptsChart.tsx`

5. **F5 — RepoSelector listbox pattern**
   - Simplify: replace `role="listbox"`/`role="option"` with plain list of buttons (drop ARIA listbox pattern since arrow key navigation is not implemented)
   - OR implement full arrow key navigation with `aria-activedescendant`
   - File: `src/components/shell/RepoSelector.tsx`

### MEDIUM priority fixes (WCAG Level AA):

6. **F6 — SessionFilterBar search label**
   - Add `aria-label="Search sessions"` to the search input
   - File: `src/components/sessions/SessionFilterBar.tsx`

7. **F7 — SessionDetail button aria-labels**
   - Add `aria-label="Navigate to prompt {pid}"` on next-recommended buttons
   - Add `aria-label="View prompt {promptId}"` on View Prompt button
   - File: `src/components/sessions/SessionDetail.tsx`

8. **F8 — Superseded/cancelled badge contrast**
   - Change `text-gray-400` to `text-gray-500` for `superseded` and `cancelled` entries
   - File: `src/constants/statusTheme.ts`

9. **F9 — Data refresh announcements**
   - Add `aria-live="polite"` visually hidden region that announces "Dashboard data loaded" on completion
   - File: `src/components/shell/AppShell.tsx`

10. **F10 — Accordion content panel roles**
    - Add `role="region"` and `aria-labelledby` to collapsible content panels
    - Ensure `id` on content panel matches `aria-controls` on trigger
    - Files: `src/components/epics/EpicCard.tsx`, `StoryRow.tsx`, `src/components/sessions/SessionCard.tsx`

### LOW priority (nice-to-have):

11. **F11 — Remove redundant onKeyDown handlers** from `EpicCard`, `StoryRow`, `SessionCard` `<button>` elements
12. **F14 — Add `<h1>` page headings** to Overview, Prompts, Sessions pages
