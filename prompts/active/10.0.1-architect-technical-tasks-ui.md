## Context for the Agent

You are the Solution Architect. Break down all Epic E2–E5 stories into detailed technical tasks with implementation specifications. Each task should include component names, file paths, props/interfaces, state management, and test requirements.

## Scope

- For each E2–E5 task, specify: component file paths, props interfaces, state hooks, data flow, test cases
- Define the UI component folder structure
- Define page/route structure
- Define shared component contracts
- Produce technical task specifications as a markdown document

## Constraints

- Do not write implementation code — produce specifications
- Ensure consistency with wireframes and UI architecture ADR
