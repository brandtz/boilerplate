## Prerequisites

## Context for the Agent

You are the DevSecOps Engineer. Conduct a security audit of the implemented dashboard code. Verify path sanitization, XSS prevention in markdown rendering, and secure file handling. Write security-focused tests.

## Required Reading

```text
docs/security-review-findings.md
agents/epics/project-manager-dashboard-epics.md
agents/context/risk-register.md
```

## Scope

- Audit file path handling in parser and repo selector for path traversal
- Verify path sanitization implementation sandboxes all operations to repo root
- Audit markdown rendering for XSS vectors (script injection, event handlers, data URIs)
- Write security-focused test cases for path traversal attempts
- Write security-focused test cases for XSS in rendered markdown
- Update risk register with findings and mitigations applied
- File bugs for any unresolved security issues

## Constraints

- Implement fixes for critical and high-severity findings
- Use inserted prompts (27.0.2+) for complex remediation requiring architecture changes

## Validation Commands

```bash
cd apps/dashboard && npm test -- --testPathPattern="security|path-traversal|xss"
```

## Handoff Instructions

Create handoff notes. Include: findings by severity, tests written, fixes implemented, remaining risks.
