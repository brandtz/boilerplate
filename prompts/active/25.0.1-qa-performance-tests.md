## Context for the Agent

You are the QA Test Architect. Create performance benchmarks for the parser and dashboard with large-scale test fixtures (300+ prompt files, 250+ handoff files). Define and enforce acceptable performance thresholds.

## Required Reading

```text
docs/test-strategy.md
agents/epics/project-manager-dashboard-epics.md
```

## Scope

- Generate large-scale test fixture (300+ prompt file stubs with valid frontmatter)
- Generate 250+ handoff file stubs
- Benchmark parser execution time
- Benchmark dashboard initial render time
- Define acceptable thresholds: parser <2s, UI initial render <3s
- Create performance regression test that can run in CI

## Constraints

- Generated fixtures must have valid frontmatter (not garbage data)
- Performance tests should be isolated from unit tests (separate test suite)

## Validation Commands

```bash
cd apps/dashboard && npm run test:perf
```

## Handoff Instructions

Create handoff notes. Include: benchmark results, threshold definitions, fixture details.
