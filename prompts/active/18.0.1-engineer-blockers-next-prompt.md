## Context for the Agent

You are the Senior Software Engineer. Build the blockers and warnings panel (blocked prompts, missing handoffs, invalid metadata) and the next prompt widget (full text viewer, copy-to-clipboard, rationale, prerequisite status).

## Required Reading

```text
docs/technical-tasks-e2-e5-ui.md
```

## Scope

- Build blockers panel: aggregate blocked prompts, missing handoffs, validation warnings from parser
- Build warnings panel: malformed frontmatter, unresolved prerequisites
- Build next-prompt widget: display prompt ID, title, rationale, prerequisite check
- Implement scrollable prompt text viewer in the widget
- Implement copy-to-clipboard for full prompt content
- Link to source file path
- Write interaction tests for copy action and prerequisite display

## Constraints

- Copy-to-clipboard must work in modern browsers without Flash or deprecated APIs
- Widget must display a clear message when no prompt is eligible

```bash
cd apps/dashboard && npm test -- --testPathPattern="blocker|next-prompt|widget"
```
