## Context for the Agent

You are the Master Agent Orchestrator. This is a review gate. Verify that all pre-implementation deliverables are complete, consistent, and approved. Summarize the state of the project and confirm readiness to enter implementation. The human sponsor must approve proceeding.

## Scope

- Verify all review documents exist and are substantive
- Check for unresolved blockers in risk register
- Check for unresolved gaps in gap analysis
- Summarize approval status from each reviewer role
- Update status-dashboard.md with current project state
- Present a go/no-go recommendation to the human sponsor

## Constraints

- Do not generate implementation code
- Do not proceed to implementation without explicit sponsor approval
- This prompt requires human interaction
